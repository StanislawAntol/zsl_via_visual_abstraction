Help Us Find Photographs of Sentences

We want you to use the Internet to find 20 different photographs that depict the same meaning/story as described by the sentence. Each image should have exactly 2 people. To perform this task, we expect it to take you about 2 hours (depending on the sentence) to accumulate a satisfactory image collection.

Please copy and paste the URLs (i.e., web addresses) that point directly to the images. Please do not provide links to other webpages, etc. The links should point directly to the image (i.e., it should end in .jpg, .jpeg, .png, .gif, .bmp, .tif, .tiff and not in .htm/.html). Feel free to use any resources such as Flickr, Google, and Bing. Please do not find drawings and only submit actual photographs.

For example:

Example Sentence Prompt:
One person is walking behind another person.		 Example Found Image:
